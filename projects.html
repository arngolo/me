IoF50 - Imagery on the Fly
Devised for Geospatial technology users, this web application allows spatial analysis on the fly using GEE python API. It reduces user's computational costs, is memory efficient and users do not require a high storage capacity since the output result is primarily stored in local then uploaded to google cloud storage (GCS).
